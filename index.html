Simple setup for csv log file on to kibana3

Download

  1. logstash-1.4.2
  2. kibana-3.1.0
  3. elasticsearch-0.90.11 s Steps for dataflow
    • Start the elastic search.
    • Open terminal to command  »elasticsearch-0.90.11/bin> elasticsearch
  • Logstash
    • Create file Grok file based on csv data format of logfile. name it as .config file(eg: test.config)
    • Save the file as .config in logstash-1.4.2/bin folder
    • Open terminal to command »logstash-1.4.2/bin>logstash agent -f test.config
  • Kibana
    • Open terminal to command »logstash-1.4.2/bin>logstash web
